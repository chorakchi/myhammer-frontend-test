### Setup

```sh
    $ git clone https://github.com/chorakchi/myhammer-frontend-test.git
    $ cd myhammer-frontend-test
    # install dependencies
    $ npm i           
    # serve develpement production
    $ ng serve        
```

### Build

```sh
    # local deployment
    $ ng bulid for            
    # deploy as production
    $ ng bulid --prod  

    # adress of APIs set in environments > 
    # environment.ts & 
    # environment.prod.ts   
```

### Test

```sh
    # run unit tests
    $ ng test            
    # run end to end test
    $ ng e2e             
```
